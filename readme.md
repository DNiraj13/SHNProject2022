This is a Maven project based on the POM and JUnit framework.
- feature file can be found in the test/resources/cucumber folder
- In pom.xml all the dependencies are declared/defined
- For global variable like URL of the application, properties file is used
- Log for the tests are generated in the log folder
- Extent HTML reports can be found in the test-output folder
- Recordings of the test can be found in the testRecordings folder
- Test can run either directly via feature file or runner class or using maven command
